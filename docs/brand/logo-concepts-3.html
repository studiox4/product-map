If they designed it / eight master takes

Same brief — P + M as a Gantt chart — run through eight legendary design minds. Each keeps the bar-monogram idea but bends it to that designer's philosophy: Rand's primary-color wit, Bass's cut-paper warmth, Vignelli's rational grid, Leader's hidden arrow, Draplin's heavy build, Scher's typographic shout, Bierut's single smart idea, Janoff's friendly notch.
