Product Map / logo system

One idea, four directions. The mark turns P and M into stacked, staggered rounded bars — read it as a monogram, read it as a Gantt chart. Each card pairs the standalone icon with the full wordmark lockup, where the P and M carry the accent and the rest of the word stays ink-black.

Concept: each glyph is drawn on a 7-row grid, then every horizontal run of cells is merged into one rounded bar — the same primitive a Gantt chart uses. Staggered start positions per row give the timeline read. Swap palette / radius / bar height in CONFIG below to retune. All four marks render from the same generator, so the family stays consistent.
